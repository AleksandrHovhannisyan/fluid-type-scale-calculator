How it works

A {% link "https://every-layout.dev/rudiments/modular-scale/" %}modular type scale{% endlink %} consists of a baseline font size and proportionally smaller and larger font sizes. Design systems traditionally used static font sizing, where each step has a fixed font size at every viewport width.

By contrast, in a fluid type scale, each modular step has a minimum, maximum, and variable font size. We can leverage {% link "https://developer.mozilla.org/en-US/docs/Web/CSS/clamp()" %}CSS clamp{% endlink %} and viewport width (vw) units to generate a set of font size variables that scale linearly between their minimum and maximum sizes, with mathematical precision.

This is done with linear interpolation by solving the equation y = mx + b, which corresponds to the preferred value for a CSS clamp declaration.
