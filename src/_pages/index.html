Generate font size variables for a fluid type scale. Grab the output CSS and drop it into any existing design system.

How it works

A modular type scale consists of a baseline font size and proportionally smaller and larger font sizes. We can leverage CSS clamp to generate a set of font size variables that scale linearly between a minimum and maximum font size. This involves solving the equation y = mx + b to compute the preferred value for clamp.
